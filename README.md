## 🛠️ How to Host on GitHub Pages

1. Create a new GitHub repository named `kabilesh-s.github.io`
2. Upload all files from this folder into the repo root
3. Go to **Settings → Pages → Source → Deploy from branch → main**
4. Your site will be live at `https://kabilesh-s.github.io`
